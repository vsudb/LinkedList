| Операция                          | Список на указателях     | Список на массиве (индексы) |
| --------------------------------- | ------------------------ | --------------------------- |
| Доступ к i-му элементу            | O(n)                     | O(n)                        |
| Последовательный обход            | ❌ медленный (cache miss) | ⚡ быстрее (cache-friendly)  |
| Поиск элемента                    | O(n), медленно           | O(n), быстрее               |
| Вставка в начало                  | O(1)                     | O(1)                        |
| Вставка в середину (по итератору) | O(1)                     | O(1)                        |
| Вставка в середину (без позиции)  | O(n)                     | O(n)                        |
| Удаление (по итератору)           | O(1)                     | O(1)                        |
| Удаление (по значению)            | O(n)                     | O(n)                        |
| Выделение памяти                  | ❌ много `new`            | ✅ один массив               |
| Локальность памяти                | ❌ плохая                 | ✅ хорошая                   |
| Cache performance                 | ❌ плохая                 | ⚡ хорошая                   |
| Переаллокация                     | нет                      | иногда нужна                |
| Фрагментация памяти               | ❌ высокая                | ✅ низкая                    |
| SIMD / оптимизация                | ❌ почти нет              | ✅ возможно частично         |
| Использование памяти              | ❌ больше (указатели)     | ✅ компактнее                |
